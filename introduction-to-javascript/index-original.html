JavaScript

What is JavaScript?

ECMAScript , also known as JavaScript, is an incredibly versatile programming language used to introduce interactivity to web content. It has a wide range of applications, from webpage & app design to creating web games and artworks. Whilst it’s possible to use JavaScript for declarative programming, it’s primarily used as an Imperative language – where developers outline each step the computer must take to achieve the desired results. JavaScript is used for both client-side and server-side web development – so it’s a valuable language to learn! It’s also what we call an Object-oriented language. We’ll talk a bit more about that means later.

JavaScript Versions & Standards

  • Like with the other languages we’ve covered already, JavaScript receives version updates that are released every year or so.
  • You may have heard of ES6 – this version of JS came out in 2015. After this, versions were named by year (e.g. ECMAScript 2016, ECMAScript 2017, etc.)
  • Some features available in newer versions may not work in older versions of web browsers. It’s wise to use a version that’s at least a couple of years old, unless you really need a specific new feature.

The .js file

  • Just like with CSS, JavaScript can be inserted straight into HTML using the <scriptd> tag, but it’s bad practice.
  • Instead, create a .js file somewhere in your project directory, then add a link to it in your HTML within the <head> of the page.

Variables

  • String
  • Number
  • Boolean
  • Bigint
  • Object
  • Symbol
  • Undefined
  • Null
  • When we're programming, we inevitably come across values or data we'll need to use again later.
  • We can store these bits of data to call upon in other parts of our script. We call these stored values Variables.
  • There are 8 types of data available to us in JavaScript, and any of them can be made into a variable.

Declaring a variable

let myName;
let myAge;
let isMarried;

Note that each line is ended with a semicolon. All JS instructions should be written this way.

It’s good practice in JS for variable names to be written in camelCase.

  • We have to declare our variable in order to use it. We do this by using the let keyword, followed by our variable’s name.
  • Unlike a lot of other programming languages, you don’t have to declare a variable’s datatype in JavaScript. That’s because we can switch a variable’s type whenever we like.

Initialising a variable

let myName = "Hannah-Louise"; 
let myAge = 29;
let isMarried = false;
  • Now we have variables, we need to give them values. This is what we call Initialising the variable.
  • To make things easier, we can declare and initialise a variable at the same time, as shown here.
  • You may have seen the var keyword used before to declare variables; this is an outdated keyword and it's never recommended for use.

Updating a variable

let myName = "Hannah-Louise"; 
let myName = "Hannah-Louise"; 
myName = "Hannah";
myName = null;
  • Once we've initialised a variable, we can update it pretty easily simply by giving it a new value further down our code.
  • Remember, JavaScript is very flexible with data types, so we can update our variable with a different data type if we need to.

Constants


const MY_NAME = "Hannah-Louise";
const namesList = [“Hannah”, “Alex”, “Adam” ]; 
namesList.push(“Emily”);

We’re using CAPITAL_SNAKE_CASE for the string const above, but camelCase for the array const below. This helps remind us at a glance which constants cannot be altered throughout the code.

  • We can also store data using a Constant instead of a variable.
  • They're a bit like variables, but with a few major differences:
    • You must initialise a constant when you declare it
    • You can't give them a new value once you have initialised them
  • Whilst you can't change the value of a constant, you can change the content of a value. Whilst that's not helpful for number or string constants, it is for Objects & Arrays.

Adding Comments

let myName = "Hannah-Louise"; // Declare myName let myAge = 29;
let isMarried = false;
/*

update the isMarried variable */
isMarried = true;
//myAge = 30;
  • We can add comments to help us make our script more readable and explain our code to others.
  • We can also use it to prevent execution of code when troubleshooting or testing alternatives.
  • We can make a comment over a single line using //. Anything written after this on the line will be considered a comment.
  • For comments over multiple lines, we use /* and */ to signal the start and end of a comment.

console.log()

let myName = "Hannah-Louise"; 
myName = "Hannah"; 
console.log(myName);
  • We can use the console.log() method to help us troubleshoot and monitor our code.
  • Activating Developer tools in a web browser will bring up the console so we can see what's happening on the webpage.

Basic Maths in JavaScript

let myInt = 7;
let myFloat = 24.62;

typeof myInt;
typeof myFloat;
  • At this stage, there's two main types of number we need to be aware of.
  • Integers are whole value numbers (e.g. 1, 100, 365, etc.)
  • Floating-point numbers (Floats) are numbers with decimal points and places (e.g. 1.5, 3.1415926, 6.9, etc.)
  • Unless your number is extremely large, all your numbers will come under the number data type in JS.
  • Number values are written without speech marks. Otherwise, JS will read it as a String!
let myInt = 7;
let myNumber = "7";
myNumber = Number(myNumber) + 5;

myNumber;
  • There are a huge number of different methods we can use with number values in JS.
  • We can also convert other data values into number types using the Number() constructor.
  • This is useful when converting values entered on web forms – which are strings – into numbers we can calculate with.
  • In this example, if we did not convert myNumber, the result would be "75". Once we do, the output becomes 12.

Arithmetic Operators

  • The way we make calculations in JavaScript is by using Arithmetic Operators; most of these will already likely be familiar to you.
  • The eModulo operator will divide the number on the left by the number on the right, returning the remainder value if the two numbers cannot be divided exactly.
  • Exponent performs a exponentiation or 'to the power of' calculation, which raises the number on the left to the power of the number on the right. In our example, we raise 5 to the power of 2; 5 squared is 25.
